# End-to-End Operations Analytics Portfolio

This portfolio contains three applied analytics projects for Supply Chain Analyst, Operations Analyst, Demand Planning, Business Analyst, and Management Trainee roles.

## Projects

### 1. Demand Planning Dashboard
Focus: forecast accuracy, MAPE, bias, demand vs supply, inventory days, backlog, service level, and stockout risk.

### 2. Logistics Network Optimization Case
Focus: station utilization, route cost, transit time, overloaded sites, rebalancing scenarios, and before/after cost-service comparison.

### 3. Business KPI Reporting Automation
Focus: SQL aggregation, Python chart automation, weekly KPI report generation, GMV/revenue/orders, fulfillment, SLA, returns, cancellations, and action items.

## Skills demonstrated
- Python data analysis
- SQL business queries
- KPI design
- Supply chain and operations logic
- Forecasting metrics
- Dashboard/reporting thinking
- Business storytelling

## How to use
Each folder is a standalone mini project. Open a project folder, install the requirements, and run the Python script. Generated charts are saved into the local `outputs/` folder.
